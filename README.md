# 如何使用

1. git clone https://github.com/kangshaojun/flutter-book.git

2. 创建一个flutter的示例工程

3. 拷贝每个示例的main.dart pubspec.yaml及资源文件至示例工程对应目录

4. 执行flutter packages get后，运行项目即可
